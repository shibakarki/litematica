What is Litematica?

Litematica is a powerful mod for Minecraft that helps you load, create, and replicate schematic builds. It's perfect for creative and technical builders!

Prerequisites

Before using Litematica, make sure you have the following installed:

  • Minecraft Forge or Fabric Loader
  • Litematica Mod
  • MaLiLib (Required Library)

Setting Up Litematica

Follow these steps to install and set up Litematica:

  1. Install Fabric Loader from the official website.
  2. Download the Litematica and MaLiLib mods for your version.
  3. Place the mod files in the `.minecraft/mods` folder.
  4. Launch Minecraft using the Fabric loader.

How to Use Litematica

Loading a Schematic

Press M to open the menu and select a schematic file. Adjust the position and start building block by block!

Adjusting the Schematic

Move, rotate, or modify the schematic using the placement tool. Adjust layers for better visibility while building.

Creating Your Own Schematic

Select two corners of your build and save the area as a schematic to share or reuse later.

Tips for Using Litematica

  • Customize keybinds for a better experience.
  • Use the material list to gather resources in Survival Mode.
  • Organize schematics into subfolders for easy access.
